# TerraContol.Arduino
## Simple Arduino sketch for terrarium controlling.

###### Parts:
- Arduino Uno
- DHT11
- HC-05/HC-06
- 1 x RGB Led
- PC fan

Sketch is using DHT11 Library.
> Don't forget to add this [library](https://github.com/adidax/dht11) to the Arduino IDE.
